# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue.

Instead join this server and open a ticket (WIP): 

- **https://fluxer.gg/uzJJ2tOq**
  
We will acknowledge your report as soon as possible and work with you to understand and address the issue before any public disclosure.

## What to Include

When reporting, please provide as much detail as you can:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

Thank you for helping keep the Fluxer bot secure
